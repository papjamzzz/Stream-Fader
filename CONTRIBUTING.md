# Contributing to StreamFader

Thanks for your interest. StreamFader is an open project and practical contributions are welcome.

## What's Welcome

- Bug reports via GitHub Issues
- Feature suggestions (check the roadmap first)
- Pull requests for fixes, improvements, or new data sources

## Ground Rules

- Do not submit API keys, tokens, or credentials of any kind
- Do not submit scraped data, private analytics exports, subscriber lists, or cache files
- Keep PRs focused — one thing at a time is easier to review
- If you're adding a feature, open an issue first so we can align before you build

## Getting Started

```bash
git clone https://github.com/papjamzzz/Stream-Fader.git
cd Stream-Fader
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```
